# Privaatsuspoliitika — Jutusta: Loe eesti keeles

_Viimati uuendatud: 2026-06-11_

See Chrome-laiendus loeb valitud teksti eesti keeles ette. Allpool on kirjeldatud, milliseid andmeid see töötleb.

## Mida laiendus töötleb
- **Sinu valitud tekst.** Kui käivitad ettelugemise, saadetakse valitud tekstilõik Jutusta API-le (`https://jutusta.ee`) tõlkimiseks ja kõneks muutmiseks. Tekst saadetakse ainult selle toimingu tegemiseks ega seda ei säilitata laienduse poolt.
- **Sinu API võti.** Sinu Jutusta API võti salvestatakse **ainult lokaalselt** sinu brauseris (`chrome.storage.local`) ja saadetakse päringutes ainult Jutusta API-le autentimiseks.

## Mida laiendus EI tee
- Ei kogu ega edasta analüütikat, lehitsemisajalugu ega isikuandmeid.
- Ei müü ega jaga andmeid kolmandatele osapooltele.
- Ei saada andmeid kuhugi mujale peale `https://jutusta.ee`.
- Ei sisalda jälgijaid ega reklaami.

## Kolmas osapool
Valitud tekst töödeldakse Jutusta teenuses (jutusta.ee) tõlke ja kõnesünteesi tegemiseks. Vt Jutusta enda tingimusi: https://jutusta.ee

## Andmete säilitamine
Ainus laienduse salvestatav andmeüksus on API võti, mis püsib sinu brauseri lokaalses salvestuses kuni sa selle eemaldad (seadetest või laienduse deinstallimisel).
